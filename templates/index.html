Spotify NUI

Allows for Spotify user experience to be controlled naturally through common hand gestures. This is the server that
bridges the connection to the Spotify API using OAuth 2.0 Verification.

Token Generation

A token is needed to use of the Spotify API.
The Flask server will save the token within a text file.
After an hour, the token will expire. Use the button below
to generate another one as needed.
